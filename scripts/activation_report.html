Activation Delta Analysis Report

13 examples across 8 memories, 40 transformer layers, 5120-dim embeddings

1. The Map — Where Each Injection Attempt Lands

What this shows: Each dot is one experiment — we injected a set of character details via KV cache prefix, then captured how much the model's internal state changed compared to no injection. UMAP squashes the 200,000-dimensional activation change into 2D so we can see if similar injections land near each other.

Bigger dots = more details successfully transferred into the output. X markers = weak transfer (<30% of details appeared). Same color = same character/memory.

2. The Key Question — Does Rewording the Same Facts Give the Same Signal?

What this shows: For memories where we tried multiple phrasings of the same facts (e.g. bullet points vs prose vs narrative), this measures whether those different wordings produced the same direction of activation change.

Green bar (>0.7) = "Yes, these phrasings reliably produce the same internal signal" — the model is encoding the meaning, not the exact words.
Red bar (<0.3) = "No, different wordings produce different/opposite signals" — either the phrasing failed to inject, or the model is tracking surface tokens.

The percentages show how many details each form actually transferred.

3. Where in the Model Does the Change Happen?

Left plot: Each line is one injection attempt. The x-axis is the transformer layer (0 = input, 39 = output). The y-axis is how much that layer's output changed. Faded lines = weak injections (<30% detail transfer).

Notice the massive jump at layer ~19 (48% depth) — that's where the model transitions from "processing syntax" to "reasoning about meaning."

Right plot: PCA tells us how many independent "directions" the activation changes span. If one component explains 90%, all injections are pushing in roughly the same direction — there's a universal "I've been primed" signal.

4. After Removing the Universal Signal — Can We Tell Memories Apart?

Left: Raw UMAP (dominated by the universal "you were primed" component — 91% of variance).
Right: After subtracting that universal component. If same-colored dots cluster together here, it means each memory has a unique fingerprint in the remaining 9% of signal. That fingerprint is what a translator MLP would learn to convert into weight changes.

What we want to see: Same-color dots clustering together on the right plot, especially for memories with multiple surface forms (same color, different numbers).

Summary

What the data says so far:
• 91% of all activation change is a single universal direction — the model has one dominant "I was given context" axis
• Elara's 3 surface forms align at 0.90 cosine similarity — proof that the model encodes meaning, not tokens
• Surface forms that fail to transfer details also fail to align — the metric is accurately measuring real semantic injection
• We need more data (100+ examples) with more surface form variations to build training data for the MLP translator
